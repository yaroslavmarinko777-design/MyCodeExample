# Dokumenten-Manager 

Dies ist eine Konsolenanwendung zur Verwaltung von Dokumentennamen.

### Funktionen:
- Hinzufügen von Dokumenten
- Liste aller gespeicherten Dokumente 
- Suchfunktion
- Speicherung in einer lokalen Textdatei

### Installation:
Sie können das Programm in Visual Studio starten, zum Beenden geben Sie 4 im Programm ein.
