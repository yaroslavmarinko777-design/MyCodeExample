# MyGame - Mini C# Windows Forms Spiel (verbesserte Version mit Bezeichnungen im Code)

Das ist ein kleines C# Windows Forms Projekt.

Der Quellcode der Anwendung ist vollständig vorhanden und kann in Form1.cs eingesehen werden.
Der Design-Teil (Form1.Designer.cs) ist leider verloren gegangen, daher kann die Benutzeroberfläche in Visual Studio nicht visuell angezeigt werden.

## Nutzung

Das Spiel kann direkt in Visual Studio geöffnet und gestartet werden.
﻿﻿Der Code zeigt die komplette Logik der Spielbewegungen, Hintergrundanimation und Ereignisbehandlung (Tastatur und Maus).

## Hinweis

Das Projekt dient dazu, meine Programmierkenntnisse in C# zu zeigen. Die visuelle Darstellung der Form ist nicht verfügbar, nur der Code ist enthalten.
