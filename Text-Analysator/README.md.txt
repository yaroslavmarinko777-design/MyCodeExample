# Analysator der Texte 

Eine kleine Programm von mir, inspiriert von typischen Aufgabenstellungen für Anwendungsentwickler
Ziel: Meine Struktur und Grundlagen zu zeigen

Diese kleine Konsolenprogramm analysiert Teste, die eingegeben werden:
- Die Gesamtzahl der Wörter
- Die Häufigkeit jedes Wortes im Text

Passt perfekt für Analyse von Diktaten
