# MySecurity_System

Mein kleines Sicherheitssystem für Benutzer:

# Kernfunktionen
﻿﻿- Identitätsprüfung: Benutzername und Sicherheitscode getrennt.
﻿﻿- 30-Sekunden-Limit: Sobald der Nutzer eingegeben wird, startet ein Timer. 
- Bei Zeitüberschreitung wird der Zugriff gesperrt.
﻿﻿- Präzision: Nutzung des time-Moduls für Berechnung der Eingabedauer.

# Technische Details
﻿﻿- Datenstruktur: Verwaltung der Nutzer in einem Dictionary (Key-Value-Paare).
﻿﻿- Logik: Verschachtelte if-else-Abfragen zur Validierung von Zeit und Code.
﻿﻿- User-Experience: Rückmeldungen bei Fehlern (Zeitlimit oder falscher Code).
